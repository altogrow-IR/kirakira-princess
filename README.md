# きらきらプリンセス

5歳の女の子が、毎日の「おやくそく」を楽しく続けるための React + TypeScript + Vite 製Webアプリです。
「できた！」を押すと、はなまるが増え、プリンセスランク・たからもの・シールちょうに反映されます。

## 使い方

1. `きょう` 画面で、おやくそくの `できた！` を押します。
2. もう一度押すと、その日の記録を戻せます。はなまるは0未満になりません。
3. `プリンセス` 画面で、現在のプリンセスランクを確認できます。
4. `たからもの` 画面で、はなまる数に応じて開いたものを確認できます。
5. `シール` 画面で、がんばった日のシールを確認できます。
6. `パパ・ママ` 画面で、おやくそく、プリンセス画像、シール画像を設定できます。

## 開発方法

```bash
npm install
npm run dev
```

## ビルド方法

```bash
npm run build
```

ビルド結果は `dist` に出力されます。

## GitHub Pages公開方法

`vite.config.ts` の `base` は、公開するリポジトリ名に合わせて変更してください。

```ts
export default defineConfig({
  plugins: [react()],
  base: "/kirakira-princess/",
});
```

リポジトリ名が `my-princess-app` の場合は、`base: "/my-princess-app/"` にします。

## データ保存

このアプリはログイン機能を使わず、ブラウザの `localStorage` に保存します。
保存キーは `kirakira-princess-app-v1` です。

保存される主な内容は以下です。

- おやくそく
- 日付ごとの達成記録
- 合計はなまる数
- シール記録
- プリンセス画像
- シール画像

画像は Data URL として `localStorage` に保存されます。大きい画像をたくさん登録すると、ブラウザの保存容量を圧迫する可能性があります。画像は 512x512px 程度を目安にしてください。

## 実装メモ

- 旧データに `star`、`earnedStars`、`requiredStars`、`totalStars` がある場合は、読み込み時に新しいはなまる用データへ移行します。
- 不正な保存データがある場合は、アプリが落ちないよう初期データへ戻します。
- 削除やリセットには確認ダイアログを表示します。
- シール画像は獲得済みシールに直接保存せず、表示時に現在のシール設定を参照します。
